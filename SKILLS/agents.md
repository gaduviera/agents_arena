# 🤖 Agent-Specific Skills

Habilidades individuales activadas por cada plugin de agente.

## Codex
- `codex:rescue`: Recuperación de errores en tiempo de ejecución de Python.
- `codex:codex-cli-runtime`: Entorno optimizado para ejecución de scripts complejos.
- `codex:gpt-5-4-prompting`: Técnicas de prompting avanzado para generación de código.

## Gemini
- `gemini:rescue`: Depuración asistida por razonamiento multimodal.
- `gemini:gemini-cli-runtime`: Bridge para ejecución de tareas en el sistema operativo.
- `gemini:gemini-prompting`: Optimización de prompts para modelos de Google.

## Qwen
- `qwen:rescue`: Manejo de límites de contexto y errores de tokenización.
- `qwen:qwen-cli-runtime`: Ejecución ligera optimizada para Qwen.
- `qwen:qwen-prompting`: Patrones específicos para la familia de modelos Qwen/DashScope.

## Ollama
- `ollama:rescue`: Recuperación ante fallos de conexión local o caída de modelos.
- `ollama:run-model`: Gestión dinámica de carga/descarga de modelos en memoria.
- `ollama:setup`: Configuración automática de variables de entorno para modelos cloud (GLM, Minimax, etc).
